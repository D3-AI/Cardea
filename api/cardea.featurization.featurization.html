generate_feature_matrix(es, target, cutoff, verbose=True)[source]¶

Calculates a feature matrix and features given in Featurization object. Args: es: A featuretools entityset that holds injested data. target: A string of the target entity name. cutoff: A pandas dataframe that indicates cutoff_time for each instance. verbose: A boolean indicator of verbose option. Returns: A pandas dataframe of the calculated matrix.
