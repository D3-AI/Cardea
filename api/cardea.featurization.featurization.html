generate_feature_matrix(es, target, cutoff, verbose=True)[source]¶

Calculates a feature matrix and features given in Featurization object.

Parameters

  • es (featuretools.EntitySet) – An already initialized entityset. Required if entities and relationships are not defined.

  • target (str) – A string of the target entity name.

  • cutoff (pandas.DataFrame) – Specified times at which to calculate the features for each instance.

  • verbose (bool) – An indicator of verbose option.

Returns

  • The generated feature matrix.

  • List of feature definitions in the feature matrix.
