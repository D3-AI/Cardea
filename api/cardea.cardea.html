load_data_entityset(folder_path=None)[source]¶

Returns an entityset loaded with .csv files in folder_path.

Load the given dataset within the folder path into an entityset. The dataset must be in a FHIR structure format. If no folder_path is not passed, the function will automatically load kaggle’s missed appointment dataset.

Parameters

folder_path (str) – A directory of all .csv files that should be loaded.

Returns

An entityset with loaded data.

Return type
